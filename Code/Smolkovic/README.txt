Compilation instructions: 
	run "make" to produce an executable named "LAB"


Usage instructions:
	./LAB < input.file	


where the input.file contains:
	first line 
		N  // number of taxons

	N(N-1)/2 lines 
		a b d  // taxon a id, taxon b id, d = distance between a and b -> d(a,b)
